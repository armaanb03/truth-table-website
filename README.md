# Truth Table Generator

This app takes semantic logic formulas that are well formed and outputs the truth table for the inputted formula using a recursive JavaScript algorithm. Developed using ReactJS.

Domain: https://armaanb03.github.io/truth-table-website/

Input must adhere to the rules of a well formed formula.

## Operators

The operators for this application are:


### AND: 
&
### OR: 
| , ||
### IMPLICATION: 
-->
### EQUIVALENCE: 
<->

### NEGATION:
~
